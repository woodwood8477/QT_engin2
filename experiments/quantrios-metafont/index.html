This version is monoline: glyphs are described as center paths with one shared pen width. The goal is not an outline font yet, but a geometric centerline source that can later accept width, tension, aperture, and audio-reactive deformation.

u#1pt
sw#.64pt
cap#8pt
xh#5.10pt
% QuanTrios.mf / v3
mode_setup;
font_identifier := "QuanTriosMono";
font_size 10pt#;

u# := 1pt#;
sw# := .64pt#;
cap# := 8pt#;
xh# := 5.10pt#;

def qt_pen =
  pickup pencircle scaled sw
enddef;

beginchar("o", 5.85u#, xh# + overshoot#, 0);
  qt_pen;
  draw fullcircle
    xscaled 3.64u
    yscaled (xh + 2overshoot)
    shifted (2.92u, .50xh);
endchar;

beginchar("n", 6.05u#, xh#, 0);
  qt_pen;
  draw (1.02u, 0)
    -- (1.02u, xh)
    .. controls (2.40u, xh + .12u)
    and (4.92u, xh - .10u) .. (4.92u, 2.92u)
    -- (4.92u, 0);
endchar;

Next implementation direction

The source should be developed as a monoline font model first. Once the center paths read as letters, the QT engine can attach variable stroke profiles, Gaussian softness, and sound-driven deformation without destroying the glyph structure.
