Adjust the parameters below. This is a live renderer for checking the monoline center-path model. TERM is critical: round caps make it look like rounded gothic; flat caps move it closer to a geometric sans skeleton.

% QuanTrios.mf / v3
mode_setup;
font_identifier := "QuanTriosMono";

u# := 1pt#;
sw# := .64pt#;
cap# := 8pt#;
xh# := 5.10pt#;

% Live page controls:
% SW    -> pen width
% WDTH  -> horizontal scale
% XH    -> lowercase height
% ARCH  -> curve tension / shoulder depth
% SPACE -> sidebearing and tracking
% TERM  -> cap style: round / flat / square

Current design issue

The readable range is currently heavy and wide, which means the skeleton is too soft at normal weight. The next source update should introduce flat terminals, stronger straight segments, and letter-specific sidebearings before adding any Futura-like optical correction.
