This version is no longer a handwritten-looking centerline sketch. The METAFONT source now describes a filled, modular geometric wordmark. It is still a prototype, but the drawing model is closer to a usable QT logo/font system.

% QuanTrios.mf / v2
mode_setup;
font_identifier := "QuanTriosMeta";
font_size 10pt#;

u# := 1pt#;
stem# := .92pt#;
cap# := 8pt#;
xh# := 5.15pt#;
desc# := 1.15pt#;

def qt_rect(expr l, b, r, t) =
  fill (l,b)--(r,b)--(r,t)--(l,t)--cycle
enddef;

def qt_ring(expr cx, cy, rx, ry, thick) =
  fill fullcircle xscaled (2rx) yscaled (2ry) shifted (cx,cy);
  unfill fullcircle xscaled (2(rx-thick)) yscaled (2(ry-thick)) shifted (cx,cy)
enddef;

beginchar("Q", 8.4u#, cap# + overshoot#, desc#);
  qt_ring(4.10u, .50cap, 3.00u, .50cap + overshoot, stem);
  qt_slant_rect(5.28u, .88u, 7.20u, -desc, stem * .94);
endchar;

Next implementation direction

This METAFONT file should now be treated as a geometric source model, not a final font. The next QT-engine step is to port the same modular glyph logic into JavaScript, then expose width, aperture, stroke, and tension as audio-reactive parameters.
