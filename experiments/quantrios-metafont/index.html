This renderer is now testing a Futura-like center skeleton, not a generic rounded monoline. It still uses strokes, so it will never equal Futura's filled outlines, but the target is the same letter logic: circular bowls, compact r, short Q tail, and flatter terminals.

% QuanTrios.mf / v4
font_identifier := "QuanTriosMono";

u# := 1pt#;
sw# := .82pt#;
cap# := 8pt#;
xh# := 5.05pt#;

% Futura-skeleton monoline test:
% Q tail short
% a = circular bowl + tangent stem
% r = compact shoulder
% s = less script-like, tighter waist

Known limitation

A monoline source can approximate Futura's center skeleton, but Futura's final identity comes from filled outlines, not a stroked path. The useful goal here is not a copy of Futura; it is a Futura-informed skeleton that can later support sound-driven stroke profiles.
